# fagans-plantilla-modif

Repositorio para gestionar los cambios en la plantilla de Shopify.

## Estructura de directorios

| Directorio  | Descripción |
|-------------|-------------|
| `assets/`   | Archivos estáticos: CSS, JavaScript, imágenes y fuentes |
| `config/`   | Archivos de configuración del tema (`settings_schema.json`, `settings_data.json`) |
| `layout/`   | Plantillas de diseño principales (`theme.liquid`, `password.liquid`) |
| `locales/`  | Archivos de traducción/internacionalización (JSON) |
| `sections/` | Secciones reutilizables de Shopify (Liquid) |
| `snippets/` | Fragmentos de código Liquid reutilizables |
| `templates/`| Plantillas de página para cada tipo de recurso (producto, colección, etc.) |
